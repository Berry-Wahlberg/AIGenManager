## 修复欢迎界面的多语言支持

### 问题分析
WelcomeWindow.xaml 中以下元素使用硬编码文本，未使用本地化绑定：
- "Set database location and language" (第45行)
- "Please select the database file location and interface language settings." (第46行)
- "Browse..." (第56行)
- "Interface Language:" (第65行)

### 修复步骤

1. **添加本地化键值到语言文件**
   - 在 en-US.json 中添加:
     - `Welcome.Database.Title` = "Set database location and language"
     - `Welcome.Database.Subtitle` = "Please select the database file location and interface language settings."
     - `Common.Buttons.Browse` = "Browse..."
     - `Welcome.Database.InterfaceLanguage` = "Interface Language:"
   - 在 zh-CN.json 中添加对应中文翻译
   - 在 zh-TW.json 中添加对应繁体中文翻译
   - 在其他语言文件中添加对应翻译

2. **更新 WelcomeWindow.xaml 使用本地化**
   - 将硬编码文本改为 `{lex:Loc ...}` 绑定
   - 确保所有文本都使用本地化

3. **初始化欢迎界面的语言设置**
   - 在 WelcomeWindow 构造函数中，根据 SelectedLanguage 初始化 LocalizeDictionary.Instance.Culture
   - 确保欢迎界面启动时语言设置就生效

### 预期结果
- 欢迎界面所有文本都支持多语言
- 用户选择语言后，欢迎界面立即以新语言显示
- 与其他界面保持一致的多语言体验
