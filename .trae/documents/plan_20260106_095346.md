# 文件夹和文件名重命名计划

## 1. 重命名范围

需要将所有以Diffusion.xxx命名的文件夹和文件重命名为BerryAIGen.xxx，包括：

### 解决方案文件
- `Diffusion.Toolkit.sln` → `BerryAIGen.sln`

### 项目文件夹和项目文件
- `Diffusion.Civitai` → `BerryAIGen.Civitai`
- `Diffusion.Common` → `BerryAIGen.Common`
- `Diffusion.Data` → `BerryAIGen.Data`
- `Diffusion.Database` → `BerryAIGen.Database`
- `Diffusion.Github` → `BerryAIGen.Github`
- `Diffusion.Scanner/Diffusion.IO.csproj` → `BerryAIGen.IO/BerryAIGen.IO.csproj`
- `Diffusion.Scripting` → `BerryAIGen.Scripting`
- `Diffusion.Toolkit` → `BerryAIGen.Toolkit`
- `Diffusion.Updater` → `BerryAIGen.Updater`
- `Diffusion.Video` → `BerryAIGen.Video`

### 其他需要更新的内容
- 所有项目文件中的命名空间引用
- 所有项目的引用关系
- 所有代码文件中的using语句
- 所有代码文件中的类型引用

## 2. 重命名顺序

1. 重命名解决方案文件
2. 更新解决方案文件中的项目引用
3. 逐个重命名项目文件夹和项目文件
4. 更新每个项目文件中的命名空间和引用
5. 更新所有代码文件中的using语句和类型引用
6. 编译项目，检查是否有编译错误
7. 进行细节检查

## 3. 详细步骤

### 3.1 重命名解决方案文件
- 将`Diffusion.Toolkit.sln`重命名为`BerryAIGen.sln`

### 3.2 更新解决方案文件
- 更新解决方案文件中的所有项目名称和路径

### 3.3 重命名项目文件夹和项目文件
- 重命名`Diffusion.Civitai` → `BerryAIGen.Civitai`
- 重命名`Diffusion.Common` → `BerryAIGen.Common`
- 重命名`Diffusion.Data` → `BerryAIGen.Data`
- 重命名`Diffusion.Database` → `BerryAIGen.Database`
- 重命名`Diffusion.Github` → `BerryAIGen.Github`
- 重命名`Diffusion.Scanner` → `BerryAIGen.IO`，并将其中的`Diffusion.IO.csproj`重命名为`BerryAIGen.IO.csproj`
- 重命名`Diffusion.Scripting` → `BerryAIGen.Scripting`
- 重命名`Diffusion.Toolkit` → `BerryAIGen.Toolkit`
- 重命名`Diffusion.Updater` → `BerryAIGen.Updater`
- 重命名`Diffusion.Video` → `BerryAIGen.Video`

### 3.4 更新项目文件
- 更新每个项目文件中的命名空间
- 更新每个项目文件中的项目引用
- 更新每个项目文件中的AssemblyName和RootNamespace

### 3.5 更新代码文件
- 使用全局替换将所有代码文件中的`namespace Diffusion.`替换为`namespace BerryAIGen.`
- 更新所有代码文件中的using语句
- 更新所有代码文件中的类型引用

### 3.6 编译和细节检查
- 编译项目，检查是否有编译错误
- 检查所有项目引用是否正确
- 检查所有命名空间是否正确
- 检查所有类型引用是否正确

## 4. 注意事项

1. 重命名操作需要按照一定的顺序进行，以确保项目能够正常编译
2. 重命名后需要更新所有相关的引用和命名空间
3. 重命名过程中需要注意大小写和拼写
4. 重命名完成后需要进行全面的编译和测试

## 5. 预期结果

- 所有文件夹和文件名都已从Diffusion.xxx重命名为BerryAIGen.xxx
- 项目能够正常编译
- 所有引用和命名空间都已更新
- 所有类型引用都已更新
